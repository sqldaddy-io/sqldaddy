## sql<>daddy.io
[sql<>daddy.io](https://sqldaddy.io/) is open-source sandbox and environment for real-time SQL experimentation

### Installation
To deploy a project locally just run make command
```
make dc_build

make dc_up
```
This is to run the Docker Composer commands
